<Icon icon="tag" iconType="duotone" /> Since [1.16.0][toolkit-1.16.0]

This aggregate combines the aggregates generated by other `max_n`
aggregates and returns the maximum values found across all the
aggregated data.


## Arguments

The syntax is:

```sql
rollup(
    agg MaxN
) MaxN
```
| Name | Type | Default | Required | Description |
|--|--|--|--|--|
| `agg` | MaxN | - | ✔ | The aggregates being combined |

## Returns

| Column | Type | Description |
|--------|------|-------------|
| rollup | MaxN | An aggregate over all of the contributing values. |
