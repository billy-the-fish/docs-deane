import { TIMESCALE_DB, TOOLKIT_LONG, CLOUD_LONG } from '/snippets/vars.mdx';

{CLOUD_LONG} includes all hyperfunctions by default, while self-hosted {TIMESCALE_DB} includes a subset of them. For
additional hyperfunctions, install the [{TOOLKIT_LONG}][install-toolkit] extension.

## Available hyperfunctions

Hyperfunctions are organized by their analytical purpose. Functions marked with ✓ in the `Toolkit` column
require an installation of {TOOLKIT_LONG} for self-hosted deployments. Functions marked with ✓ in the `Experimental`
column are still under development.

<Info>
Experimental hyperfunctions are still under active development and subject to change. When you upgrade to a new
version of {TOOLKIT_LONG}, experimental features might have changed or been removed. Be prepared to update your code
if you use experimental hyperfunctions.
</Info>
