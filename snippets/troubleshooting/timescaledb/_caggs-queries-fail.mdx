### Queries fail when defining continuous aggregates but work on regular tables

**APIs**:
- [`CREATE MATERIALIZED VIEW (Continuous Aggregate)`][create-materialized-view]

{CAGG}s do not work on all queries. For example, {TIMESCALE_DB} does not support window functions on
{CAGG}s. If you use an unsupported function, you see the following error:

```sql
      ERROR:  invalid continuous aggregate view
      SQL state: 0A000
```

For more information about supported functions in {CAGG}s, see the
[CREATE MATERIALIZED VIEW][create-materialized-view] documentation.

[create-materialized-view]: /api-reference/timescaledb/continuous-aggregates/create_materialized_view
