## Graph OHLCV data

When you have extracted the raw OHLCV data, you can use it to graph the result
in a candlestick chart, using Grafana.

1.  In Grafana, from the `Dashboards` page, click `New` and select `New dashboard`.
1.  Click `Add visualization`, then select the data source that connects to your {SERVICE_LONG} and the `Candlestick` visualization type in the top right.
1. In the `Queries` section, select `Code` and paste the query you used to get the OHLCV values:

   ```sql
   SELECT * FROM one_day_candle
   WHERE symbol = 'BTC/USD' AND bucket >= NOW() - INTERVAL '14 days'
   ORDER BY bucket;
   ```
1.  Adjust elements of the table as required, and click `Apply` to save your
    graph to the dashboard.
